== Description ==


[Demo](https://tishonator.com/plugins/tabs-block)

Tabs Block is a simple plugin that adds a Gutenberg block for adding Tabs content to your posts and pages. Fully responsive and accessible.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `tabs-block` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open for Edit or add new Post or Page
4. Click on '+' button to add new Block
5. Insert 'Tab Item' blocks
7. Save changes

Please Note: All of the 'Tab Item' blocks will be displayed as a Tab
